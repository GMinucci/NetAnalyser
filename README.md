## Installation

### Cocoapods

NetAnalyser is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NetAnalyser'
```

### SPM

Add the follow on your `Package.swift`

```swfit
.package(url: "<<repo>>", from: "0.0.1")
```
